# HDR-Pipeline
Проект строит HDR-изображение из серии кадров с разной экспозицией (один нормальный, по два засвеченных и затемненных):
- выравнивание кадров методом **MTB (Median Threshold Bitmap)**
- радиометрическая калибровка **Debevec & Malik** (оценка `g(z)` по каналам RGB)
- HDR-слияние в радианс-карту
- тонмаппинг: **Reinhard**, **Drago**, **(упрощённый) Mantiuk**
- сохранение промежуточных и итоговых изображений (в репозитории приведены только оригиналы и результат)
